# 유도단위

길이 질량 시간 온도 등의 기본단위에서 유도한것 기본단위외의 모든 단위들

## 유도단위 예1
__힘 =  질량 * (가)속도__

F = ma => [kg][ms^-2]

이를 SI단위 로 표현하면 1뉴턴이고 다음과 같다.  1N = 1kg*m/s^2

## 유도단위 예2

힘을 면적으로 나누면 압력 => N/m^2 

즉 단위면적당 받는 힘
