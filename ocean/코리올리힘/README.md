# 코리올리 효과(전향력)

북반구에서, 코리올리효과에 의해 운동방향에 대해 오른쪽으로 편향하게 된다.

남반구에서는 왼쪽으로 편향하게 된다.

이로 인해 북반구의 해류는 시계 방향, 남반구의 해류는 시계반대 방향으로 흐른다.
