# 표면파와 내부파
## 파는 서로 다른 밀도를 가진 두 유체의 경계면을 따라 형성된다. 

## 1) 표면파 =>  공기와 물의 경계면 사이에서 형성된다. 주기: 수초 

## 2) 내부파 =>  같은유체 내에서도 서로 다른 밀도층의 경계면을 따라 내부파가 형성된다. 주기: 수분 수시간
