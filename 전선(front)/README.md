# 해양 전선(front)

수온 염분 밀도 처럼 해수의 물성이 수평적으로 급격하게 변하는 부분을 해양 전선(front)이라 함.

서로 다른 수괴가 만나는 경계면임.

한류와 난류가 만나는 곳에서 전선이 발생한다.

# 대기 전선(front)

발생지가 달라 온도나 습도등 성질이 다른 두 기단이 만날때 생기는 불연속면을 전선면이라고 한다.

이 전선면이 지표면과 만나는 선을 전선이라고함. 


성질이 다른 두 기단 사이에 형성됨.


# 대기 전선의 종류

1) 한랭전선 : 한기단이 난기단 밑으로 파고 들어가는 곳에서 생기는 전선

2) 온난전선 : 난기단이 한기단의 경계면을 따라 위로 상승

3) 폐색전선 : 한랭전선과 온난전선이 합쳐진 것

4) 정체전선 : 찬 기단과 따뜻한 기단의 세력이 비슷할 때 전선이 이동하지 않는것
